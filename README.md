# CryptoPrice
iOS App: Display cryptocurrency prices fetched from Coinranking API in the last 24 hours.  Prices are formatted to display local currency.

## Built with: 
* Swift
* SwiftUI
* MVVM
* Combine
* Coinranking API
* URL Components, Codable
* Code refactoring (ongoing)
